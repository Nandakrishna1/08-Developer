# 🌐 API Landing Page (Tailwind CSS)

A responsive API marketplace/landing page built using HTML, Tailwind CSS (CDN), and Font Awesome icons.

This project showcases a modern responsive UI with navigation, API categories, blog articles, and footer sections.

[live at](http://127.0.0.1:5500/Developer/index.html)

## 📌 Features

✅ Responsive Navigation Bar

✅ Mobile-friendly design using Tailwind’s responsive utilities

✅ API Categories Section

✅ Blog Articles Section

✅ Call-to-Action (Contact & Download)

✅ Footer with Social Media Icons

✅ Hover animations and scaling effects

## 🛠️ Tech Stack

HTML5

Tailwind CSS (via CDN)

Font Awesome (Icons CDN)

## 🚀 How to Run the Project

Download or clone the repository.

Make sure all image assets are in the correct folder paths.

Open index.html in your browser.

No installation required — Tailwind and Font Awesome are loaded via CDN.

## 📱 Responsive Design

The layout adapts for:

Desktop (lg screens)

Mobile devices (hidden lg:flex utilities used)

Tailwind’s responsive classes such as:

lg:flex
hidden
lg:hidden

are used to control visibility across devices.

## ✨ Customization

You can customize:

Colors → Modify Tailwind classes (e.g., bg-blue-700, text-purple-500)

Fonts → Add Google Fonts in <head>

Animations → Adjust hover utilities (hover:scale-110)

Layout → Update Tailwind grid/flex utilities

## ✅ Conclusion

This project demonstrates how to build a clean, modern, and fully responsive API landing page using HTML and Tailwind CSS without complex setup or build tools. By leveraging Tailwind’s utility-first approach and CDN integration, the design remains lightweight, customizable, and easy to maintain.
